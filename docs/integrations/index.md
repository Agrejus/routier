## Integrations

Storage backends and framework adapters for Routier.

## Quick Navigation

## Overview

Routier provides integrations for popular frameworks and storage backends. These integrations make it easy to use Routier in your preferred environment.

If you are building a local-first app with HTTP sync, start with the [Replication Plugin](/integrations/plugins/built-in-plugins/replication/). It documents the `HttpDbPlugin`, `HttpSwrDbPlugin`, and `OptimisticUpdatesDbPlugin` compositions used for SWR and optimistic updates.
