Routier is a TypeScript npm-workspaces monorepo for a local-first reactive data toolkit. Main packages: core (schemas/expressions/plugins/utilities), datastore (collections/queryables/change tracking), react integration, storage plugins (memory/browser-storage/file-system/dexie/sqlite/pouchdb/mysql/postgresql/replication), test-utils, docs, examples. Tech stack includes TypeScript, Jest/Vitest, Rspack/Vite, npm workspaces.
